# Doctor Appointment System

A modern, responsive web application designed to facilitate booking doctor appointments efficiently. Built using Bootstrap 5 and JavaScript, this project features a polished client-side UI with advanced admin panels for managing doctors, patients, and appointments.

## Features

- Responsive design with elegant gradient backgrounds for all pages
- Client-side pages including home, doctor listing, appointment booking, user appointments, and contact support
- Secure admin login with username/password authentication modal
- Admin dashboard with overview cards and recent appointments
- Doctor management with add/delete functionality
- Appointment management with confirm and cancel options
- Patient management showing user details and appointment count
- System settings page for admin to configure preferences
- Bootstrap 5 framework with FontAwesome icons for modern UI
- Fully frontend-based, ready for backend integration




### Prerequisites

- A modern web browser (Chrome, Firefox, Edge, etc.)
- Optional: A web server for hosting (e.g., GitHub Pages, Apache, etc.)
